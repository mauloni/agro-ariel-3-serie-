# AgroForte | Produção e Meio Ambiente

A **AgroForte** é uma Landing Page moderna e responsiva voltada para o setor do agronegócio sustentável. O projeto foi desenvolvido com o objetivo de apresentar soluções que unem a alta produtividade no campo às práticas de conservação ambiental e diretrizes ESG (Governança Ambiental, Social e Corporativa).

---

## 🚀 Sobre o Projeto

O site funciona como um cartão de visitas digital e institucional, estruturado para converter visitantes em clientes em potencial. Ele aborda a necessidade atual do mercado de alinhar tecnologia agrícola, rentabilidade e respeito ao meio ambiente.

### 🎨 Design e Identidade Visual
* **Paleta de Cores:** Uso estratégico de tons de verde (`emerald`) para evocar natureza e sustentabilidade, combinado com detalhes em laranja/âmbar (`amber`) para trazer calor, energia e destacar os botões de ação (Call to Action).
* **Tipografia e Ícones:** Visual limpo e tipografia moderna sem serifa, acompanhada de ícones minimalistas que facilitam a leitura visual das seções.
* **Seções Inclusas:**
  1. **Header (Navegação):** Menu fixo (`sticky`) com logo e links rápidos.
  2. **Hero Section (Banner Principal):** Chamada de impacto com imagem de fundo, texto motivacional e botões de conversão.
  3. **Diferenciais (Cards):** Explicação dos três pilares da marca: *Produção Forte*, *Meio Ambiente* e *Inovação & ESG*.
  4. **Manifesto Institucional:** Seção com imagem integrada e um bloco de citação destacando o propósito da empresa.
  5. **Footer (Rodapé):** Direitos autorais e links para redes sociais (WhatsApp, Instagram e LinkedIn).

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas de front-end focadas em performance e desenvolvimento ágil:

* **HTML5:** Estruturação semântica de todo o conteúdo da página.
* **Tailwind CSS (v4):** Framework utilitário utilizado via CDN para estilização rápida, responsividade nativa (computador/celular) e efeitos visuais como gradientes e transições.
* **Font Awesome (v6.4.0):** Biblioteca de ícones vetoriais utilizada nos cards de benefícios e nos links de redes sociais.
* **Unsplash API (Imagens):** Utilização de bancos de imagens profissionais de alta qualidade para o background do banner e seção de manifesto.

---

## 📱 Recursos e Responsividade

* **Mobile-First Layout:** A página se adapta perfeitamente a dispositivos móveis (smartphones e tablets) e telas grandes (monitores desktop).
* **Efeitos de Hover:** Botões e cards possuem microinterações e transições suaves (`transition duration-300`) que melhoram a experiência do usuário (UX).
* **Menu Inteligente:** Elementos de navegação que se reorganizam de acordo com o tamanho da tela do usuário.

---

## 🔧 Como Executar o Projeto

Como o projeto utiliza Tailwind CSS e Font Awesome via redes de distribuição (CDN), você não precisa instalar nenhuma dependência via Node.js ou npm. 

1. Faça o clone ou baixe este repositório.
2. Abra o arquivo `index.html` diretamente em qualquer navegador web (Chrome, Firefox, Edge, etc.).
3. *(Opcional)* Se estiver utilizando o **VS Code**, recomendo abrir com a extensão **Live Server** para ver as alterações em tempo real.
